### 4. 신규 링크 생성 - 링크 생성 완료 및 링크 복사

![4.신규링크생성-링크생성 완료 및 복사하여 고객에 전달](시연 시나리오.assets/4.신규링크생성-링크생성 완료 및 복사하여 고객에 전달.png)

2. 링크 생성 시 회사명, 링크 관리자, 만료일시, 링크 이름, 고객 데이터를 입력
3. 링크 생성 버튼을 누르면 링크가 생성되고 링크를 복사한 뒤 고객들에게 링크는 전달
